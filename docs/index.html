calibrate (maxmem, maxmemfrac, maxtime)
Generate a n, r, p cost numbers based on current system performance

Parameters:

  • maxmem: The maximum amount of memory in kilobytes to use during hashing. Never uses less than 1024.
  • maxmemfrac: The maximum fraction of available memory to use during hashing. Never uses more than .5.
  • maxtime: The maximum time to spend on hashing a password. Default .2 (200ms)

Returns:

    n, r, p
