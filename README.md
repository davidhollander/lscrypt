lscrypt is a Lua C binding to the scrypt hashing library from [Tarsnap](http://www.tarsnap.com/scrypt.html) with helpers for generating and checking password strings.

##Status
Working
##Install
###luarocks
in development
###manual
    cd src/
    make
    make install
